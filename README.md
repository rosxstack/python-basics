This is my classwork in my second week of my uni. where I learn about basics of Python. 
# Python Fundamentals

A small collection of beginner Python programs demonstrating core concepts like:
- printing output
- arithmetic operators
- user input
- built-in functions (min/max)

## Projects

### Basics
- `basics/first_program.py` — simple print statements
- `basics/calculations.py` — arithmetic operations

### Math
- `math/rectangle_area.py` — calculates rectangle area
- `math/min_max_of_three.py` — finds largest and smallest of three inputs

## How to run
Make sure Python 3 is installed, then:

```bash
python basics/first_program.py
python basics/calculations.py
python math/rectangle_area.py
python math/min_max_of_three.py
